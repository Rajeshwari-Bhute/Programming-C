## 📖 About

This repository contains C programs created for learning, practice, and improving programming fundamentals.  
It includes simple logic-building examples, pattern-based programs, calculations, and basic problem-solving exercises.

## ✨ Features

- Beginner-friendly C programs
- Simple and understandable code
- Practice-based examples
- Useful for academic learning
- Good for logic building and revision

## 📂 Programs Included

- Basic input/output programs
- Arithmetic programs
- Area and calculation programs
- Simple logic-based problems
- Number-based programs
- Practice examples from `1.c` to `28.c`

## 🛠️ Language Used

- C

## 🎯 Purpose

This repository is useful for:
- Learning C programming basics
- Practicing syntax and logic
- Understanding problem-solving in C
- Building strong programming fundamentals

## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Rajeshwari-Bhute/Programming-C.git
   ```

2. Open the folder in any C editor or IDE.

3. Compile and run any `.c` file.
